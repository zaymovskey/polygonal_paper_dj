Проект на заказ

  • Cделаем проект любой скульптуры на заказ — будь то маска, фигура животного в полный рост или модель для детского творчества

  • Подготовим чертежи для изготовления из любого материала. Например, чертежи для изготовления модели из металла будут выполнены с учётом растяжения при изгибе, толщины материала и последовательности сварки.

  • Произведём тираж наборов для творчества на основе нашего проекта, сделаем упаковку в вашем фирменном стиле
